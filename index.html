O que é Git e GitHub?

Git

É um sistema de controle de versão distribuído que permite o acompanhamento das mudanças no código fonte de projetos de software. Ele foi criado por Linus Torvalds em 2005 e é amplamente utilizado por desenvolvedores para colaboração e gerenciamento de código.

GitHub

é uma plataforma de hospedagem de código-fonte que utiliza o Git como seu sistema de controle de versão. Ela permite que desenvolvedores compartilhem, colaborem e contribuam para projetos de código aberto e privados. O GitHub fornece recursos como controle de acesso, problemas (issues), solicitações de pull (pull requests) e muito mais.

Em resumo, o Git é a tecnologia subjacente que permite o controle de versão, enquanto o GitHub é uma plataforma web que torna mais fácil hospedar e colaborar em projetos de software.
